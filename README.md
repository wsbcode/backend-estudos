# 📚 Backend Estudos

Repositório dedicado aos meus estudos de **Desenvolvimento Backend**, contendo aulas, práticas e experimentos realizados durante cursos.

---

## 💻 Sobre

Este repositório reúne exercícios e projetos desenvolvidos com o objetivo de reforçar conceitos fundamentais e avançados do backend.

Aqui você vai encontrar desde a criação de APIs simples até aplicações mais estruturadas com banco de dados e autenticação.

---

## 🚀 Conteúdos Estudados

### 📌 Fundamentos Backend

* Lógica aplicada ao backend
* Estrutura de servidores
* Protocolos HTTP
* Rotas e requisições

### 🌐 APIs e Integrações

* Criação de APIs REST
* Consumo de APIs externas
* Webhooks
* Testes com ferramentas (Postman/Insomnia)

### 🧪 Práticas

* Exercícios diversos
* Testes de lógica no backend
* Pequenos projetos com APIs

### 🔐 Autenticação e Segurança

* Login e cadastro de usuários
* JWT (JSON Web Token)
* Hash de senhas
* Middlewares de proteção

### 🗄️ Banco de Dados

* SQL (MySQL / PostgreSQL)
* NoSQL (MongoDB)
* CRUD completo
* Relacionamentos
