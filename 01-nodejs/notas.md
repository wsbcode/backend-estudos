COMANDO PARA RODAR O NODE COM TS ONDE VAI ASSISTIR SEMPRE OQUE FOR FEITO
npm run dev: Ambiente de criação (rápido e automático).

COMANDO PARA CONVERTER O TS EM JS
npm run build: Preparação para entrega (gera a pasta dist).

COMANDO PARA RODAR O JS JA COMPILADO
npm run start: Código em execução real (o que o cliente usa).
